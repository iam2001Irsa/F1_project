F1 Strategy Simulator

This project is a Formula 1 race strategy simulation system. It uses historical Abu Dhabi Grand Prix data to simulate and compare race strategies.

Structure
- data/ : raw and processed datasets
- notebooks/ : data preprocessing and model training notebooks
- src/ : implementation files including main interface, results page, and simulation logic

Main Files
- main.py : user input interface
- results.py : simulation output display
- simulation.py : core simulation logic

To run:
streamlit run src/main.py (from root)
